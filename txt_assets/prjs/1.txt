In 2025, I've purchased an Ayaneo AM02 Mini PC to build a setup I've been dreaming about for a while: My very own game console that also acts as a server. Thanks to Valve, Steam can be used as a game console UI. A gamescope user session allows for a seamless boot into gaming. In the background, I've also added many services I use on daily, like a custom DNS, a VPN (two in fact), cloud storage, etc... Read more about it on the Medium article I wrote <a href='https://medium.com/@raphalbeauregard/my-dev-arcade-my-home-made-steam-machine-server-285995e99155'>here!</a> I've obtained a lot of hands-on experience with Linux server architecture, PKIs, systemctl, and several industry standard tools used on most servers around the globe.<br><br><img src=img/am02.webp style='width:100%; left:30%; box-shadow: -10px -10px 50px red, 10px 10px 50px red;'><br><p>This server is my main lab environnement, and the majority of my write-ups and other projects have been completed thanks to this setup.</p>
